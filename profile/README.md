## Overview

Acorn Advanced is an approachable yet powerful image editor from Flying Meat that provides layer-based editing, vector tools, filters, and effects in a native macOS interface that prioritizes clarity and efficiency over feature bloat. It occupies the productive middle ground between basic image viewers and complex professional suites, offering genuine editing power without the learning curve that heavyweight tools impose. The application has been independently developed for macOS since 2007, earning a loyal following among users who value thoughtful, native software.

Layer-based editing supports raster and vector layers with blend modes, opacity controls, and layer masks that enable non-destructive compositing and editing workflows that keep every element independently adjustable. Group layers organize complex compositions into manageable sections, and layer effects add shadows, glows, and other non-destructive effects that update automatically when the layer content changes. The layer system is powerful enough for complex work while remaining intuitive for casual users.

Vector tools create shapes, text, and paths that remain resolution-independent and editable alongside raster content, enabling mixed-media compositions without requiring a separate vector application. Boolean operations combine shapes into complex forms, and the shape library provides commonly needed elements. Text on path wraps typography along any curve, and vector elements can be styled with strokes, fills, gradients, and pattern fills.

The filter system applies Core Image filters with real-time preview and adjustable parameters, covering everything from color adjustments through blur and distortion effects to stylistic transformations. Filters can be chained and reordered, building complex effects from simple components. Non-destructive filter layers apply effects without modifying original pixels, enabling experimentation without commitment.

For users who need real image editing capability — layers, masks, vectors, filters — without the complexity and subscription cost of professional suites, Acorn Advanced is the image editor that delivers genuine power in an interface that respects your time and intelligence. Its native macOS design means it launches instantly and feels like a natural part of the platform.

## Performance

Acorn balances performance with system resource usage intelligently. Operations are handled in the background, keeping your system responsive for everything else. Efficient memory management and smart caching reduce overhead during extended work sessions.

Apple Silicon acceleration is fully utilized on M-series chips, delivering noticeably faster performance. On Intel machines, hardware acceleration is used wherever available.
